Patra URL Monitor — Privacy Policy

Last updated: April 29, 2026

Overview

Patra URL Monitor is a browser extension deployed by organizations as part of the Patra Endpoint Monitor platform. It is designed for enterprise endpoint security and is installed on devices managed by the organization.

Data Collected

The extension collects the following data from the browser it is installed in:

  • Page URLs — full URLs of pages visited (including query parameters)
  • Page titles — the title of each visited page
  • Tab activity — tab open, close, switch, and audio playback events
  • Browser type — which browser the extension is running in (e.g. Chrome, Edge, Brave)

The extension does not collect passwords, form inputs, file contents, cookies, browsing history beyond the active session, or any data from browser internal pages (chrome://, edge://, about:, etc.).

How Data Is Transmitted

All collected data is sent exclusively to a local agent process running on the same machine via http://127.0.0.1. No data is transmitted directly to external servers, third-party services, or the internet by the extension itself.

How Data Is Used

The local agent forwards collected data to the organization's self-hosted Patra Monitor server for the following purposes:

  • URL filtering and policy enforcement (blocking access to restricted sites)
  • Security monitoring and threat detection
  • Compliance reporting

Data Storage and Retention

Data retention periods are determined by the organization that deploys Patra Monitor. The extension itself does not persist any data locally beyond short-lived in-memory caches used for deduplication (cleared every 60 seconds).

Data Sharing

Data is accessible only to authorized administrators within the deploying organization. It is not sold, shared with, or disclosed to any third party.

User Control

This extension is deployed and managed by the organization as part of its endpoint security program. Questions about data collection scope, retention, or access should be directed to the organization's IT or security team.

Chrome Web Store Limited Use Disclosure

The Patra URL Monitor extension's use and transfer of information received from Google APIs to any other app adheres to the Chrome Web Store API Services User Data Policy, including the Limited Use requirements.

Contact

For questions regarding this privacy policy, contact the Patra Monitor development team at the email address listed on the Chrome Web Store developer profile.
